# Shoreline Wave Analysis & Generator Control

## Project Overview
This repository houses the software suite developed for the NSF Urban Shorelines Reconfiguration Research project. The system is designed to simulate, track, and analyze ocean waves in a physical tank.

More details to come regarding:
1.  Computer Vision Analysis (MATLAB)
2.  Hardware Control (Python/Arduino)
3.  TPMS Optimization

## Repository Structure

### 1. Wave Analysis (MATLAB)
* `WaveTankAnalyzer.mlapp`
* `WaveTankFunction.m`
* `createMask.m`

### 2. Hardware Control (Python & Arduino)
* Arduino Firmware (`BlueTooth.ino`, `switch.ino`)
* Python Scripts (`bluetooth_bleak.py`)

## Getting Started

### Prerequisites
* MATLAB R2022a+
* Python 3.x
* Arduino IDE
